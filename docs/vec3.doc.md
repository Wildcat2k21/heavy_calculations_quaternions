# Vec3 (rotation-mesh-core)

Минимальная математическая библиотека для 3D-векторов, используемая для графики, трансформаций и будущих операций с 3D-моделями (WASM / Rust).

---

# Структура

```rust
Vec3 {
    x: f32,
    y: f32,
    z: f32
}
```

---

# Создание вектора

```rust
let v = Vec3::new(1.0, 2.0, 3.0);
```

---

# Основные операции

## Сложение / вычитание

```rust
let a = Vec3::new(1.0, 2.0, 3.0);
let b = Vec3::new(3.0, 2.0, 1.0);

let c = a + b;
let d = a - b;
```

---

## Скалярное умножение

```rust
let v = Vec3::new(1.0, 2.0, 3.0);

let scaled = v * 2.0;
let scaled2 = v.scale(3.0);
```

---

# Геометрия

## Длина вектора

```rust
let v = Vec3::new(3.0, 4.0, 0.0);

let len = v.length(); // 5.0
```

---

## Нормализация

```rust
let v = Vec3::new(3.0, 4.0, 0.0);

let n = v.normalize(); // единичный вектор
```

---

## Скалярное произведение (dot)

```rust
let a = Vec3::new(1.0, 0.0, 0.0);
let b = Vec3::new(0.0, 1.0, 0.0);

let dot = a.dot(b); // 0.0 (перпендикулярны)
```

---

## Векторное произведение (cross)

```rust
let a = Vec3::new(1.0, 0.0, 0.0);
let b = Vec3::new(0.0, 1.0, 0.0);

let c = a.cross(b); // (0,0,1)
```

---

## Расстояние между точками

```rust
let a = Vec3::new(0.0, 0.0, 0.0);
let b = Vec3::new(3.0, 4.0, 0.0);
let d = a.distance(b); // 5.0
```

---

# Дополнительные операции

## Линейная интерполяция (LERP)

```rust
let a = Vec3::new(0.0, 0.0, 0.0);
let b = Vec3::new(10.0, 0.0, 0.0);

let mid = a.lerp(b, 0.5); // (5,0,0)
```

---

## Проекция вектора

```rust
let a = Vec3::new(2.0, 2.0, 0.0);
let b = Vec3::new(1.0, 0.0, 0.0);

let p = a.project(b); // проекция a на b
```

---

## Изменение длины

```rust
let v = Vec3::new(1.0, 2.0, 2.0);

let v2 = v.set_magnitude(10.0);
```

---

# Поддерживаемые операторы

```rust
a + b   // сложение
a - b   // вычитание
a * k   // умножение на число
a / k   // деление на число
```

---

# Мутирующие операции

```rust
v += other;
v -= other;
```

---

# Назначение

Эта библиотека используется как базовый слой для:

* 3D трансформаций
* кватернионов
* OBJ-моделей
* WASM вычислений
* визуализации в p5.js/WebGL

---

# Примечание

Все операции оптимизированы для использования в WASM (f32, без лишних аллокаций).
