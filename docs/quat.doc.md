# Quaternion (rotation-mesh-core)

Библиотека для работы с кватернионами — компактным и устойчивым способом представления 3D вращений. Используется совместно с `Vec3` и `Mat3` и подходит для WASM/JS-графики и сравнений методов вращения.

---

# Структура

```rust
Quaternion {
    w: f32,
    x: f32,
    y: f32,
    z: f32
}
```

---

# Создание кватерниона

## Произвольный кватернион

```rust id="q1"
let q = Quaternion::new(1.0, 0.0, 0.0, 0.0);
```

---

## Единичный кватернион

```rust id="q2"
let q = Quaternion::identity();
```

---

## Из оси и угла (основной способ)

```rust id="q3"
let axis = Vec3::new(0.0, 1.0, 0.0);
let angle = 1.57;

let q = Quaternion::from_axis_angle(axis, angle);
```

---

# Базовые операции

## Сложение

```rust id="q4"
let q3 = q1.add(q2);
```

---

## Вычитание

```rust id="q5"
let q3 = q1.sub(q2);
```

---

# Умножение (основная операция вращений)

```rust id="q6"
let q = q1.mul(q2);
```

### Значение:

* композиция вращений
* порядок имеет значение
* не коммутативно

---

# Норма и длина

## Квадрат нормы

```rust id="q7"
let n = q.norm();
```

## Длина (модуль)

```rust id="q8"
let m = q.magnitude();
```

---

# Нормализация

```rust id="q9"
let unit_q = q.normalize();
```

### Используется для:

* корректных вращений
* стабильных вычислений
* предотвращения накопления ошибок

---

# Обратный кватернион

```rust id="q10"
let inv = q.inverse();
```

### Используется для:

* обратного вращения
* отмены трансформации

---

# Вращение вектора

```rust id="q11"
let v = Vec3::new(1.0, 0.0, 0.0);

let rotated = q.rotate_vec3(v);
```

### Описание:

Применяет вращение к вектору без матриц.

---

# Связь с матрицами

## Преобразование в Mat3

```rust id="q12"
let m = q.to_mat3();
```

### Используется для:

* сравнения с матричным вращением
* интеграции с pipeline Mat3

---

# Пример использования

```rust id="q13"
let v = Vec3::new(1.0, 0.0, 0.0);

let q = Quaternion::from_axis_angle(
    Vec3::new(0.0, 1.0, 0.0),
    1.57
);

let v2 = q.rotate_vec3(v);
```

---

# Назначение

Кватернионы используются для:

* устойчивых 3D вращений
* композиции вращений без деградации
* интерполяции (будущее расширение)
* сравнения с матрицами вращения
* демонстрации отсутствия gimbal lock

---

# Преимущества

* стабильность при многократных вращениях
* отсутствие gimbal lock
* компактность (4 числа вместо 9)
* эффективная композиция вращений

---

# Ограничения

* менее интуитивны, чем матрицы
* требуют нормализации
* сложнее для прямой визуализации без преобразования

---

# Связь с другими модулями

```text id="q14"
Vec3       → вход/выход для вращений
Mat3       → альтернативное представление
Quaternion → основной механизм вращения
```

---

# Примечание

Кватернионы являются предпочтительным способом представления вращений в 3D-графике и игровых движках благодаря устойчивости и отсутствию потери ориентации.
